Tails Ghetto Wallet Live usb.


*This has to be done on Linux (only tested on Ubuntu)


1: Download Tails https://tails.net/install/download/index.en.html


2: Open Terminal


Run

sudo apt install git


Then

sudo apt-get install genisoimage parted squashfs-tools syslinux-utils build-essential python3-tk python3-pil.imagetk python3-pyudev


Then

git clone https://github.com/DesobedienteTecnologico/dtails

Then Change into Directory

cd dtails/


Then to Open GUI run

sudo ./dtails.py












This Should Appear!


Then Click Select Image.

Then Check the Software you want.



























Insert USB Drive to Burn the .IMG
